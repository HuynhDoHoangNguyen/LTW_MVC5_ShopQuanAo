
# LTW_MVC5_ShopQuanAo
Website quản lý shop quần áo - ASP.NET MVC5

## 🛠️ Công nghệ sử dụng
- ASP.NET MVC5 (VS 2022)
- SQL Server 2014
- Entity Framework
- Bootstrap / jQuery

## 🚀 Chức năng chính
- Quản lý sản phẩm, danh mục
- Quản lý khách hàng, giỏ hàng, thanh toán
- Đăng nhập, phân quyền tài khoản
- Quản lý đơn hàng

## 📦 Cách chạy dự án
1. Chay sql server khuyến nghị 2014
2. Update string connection trong `Web.config`
3. Build & Run
